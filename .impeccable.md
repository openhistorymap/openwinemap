# openwinemap — Impeccable design context

## Design Context

### Users
Wine-curious general public — drinkers who want to know where a name on a label came from, travellers planning a tasting route, and the wandering-atlas audience that already loves openculturemap. Sessions are exploratory ("where is Barolo, what else is around it?") rather than transactional ("book a tasting"). Device mix skews desktop for the wandering experience; mobile must remain a first-class adapted view.

### Brand Personality
Three words: **Convivial · Earthy · Storied**. The map should feel like turning the pages of a connoisseur's atlas — bottle-littered desk, candlelight, leather-bound region encyclopedia — not a wine-retail app. Emotional goal: *warmth and curiosity*, never status anxiety. We do not rate wines, we situate them.

### Aesthetic Direction
- **Inspirations**: 19th-century vineyard maps (Cassini's France, Bordeaux's *Cocks et Féret* atlases), Are.na / Cabinet for the editorial restraint, Hugh Johnson's *World Atlas of Wine* for the cartographic register.
- **Anti-references**: Vivino-style score chrome, restaurant-app gloss, "premium" black-and-gold luxury cliché, any "tasting note" UI with star ratings.
- **Visual register**: a vintner's commonplace book. Warm aged-paper light theme; deep-cellar dark theme that still reads as warm, not noir. Earth-toned but vivid — claret, gold, vine-green, terracotta — never neon, never glassy.
- **Theme**: both light and dark, toggleable. Default to **light** (paper / atlas — matches the daylight-tasting-room mood). Dark is the after-dinner cellar variant.

### Design Principles
1. **The map is the hero — the chrome is the caption.** Sidebar and controls behave like the margin notes of a printed atlas: present, useful, never competing with the cartography.
2. **Characterful typography over neutral system type.** Caprasimo display for headings, Vollkorn body for reading. Hierarchy through size and weight contrast, never gradient fills.
3. **Claret-and-gold, not red-and-yellow.** 60-30-10: paper / ink-and-borders / one claret accent. Gold reserved for highlights and the brand cluster. Vine-green for the vineyard-specific affordances.
4. **Decorative details should look hand-set.** The brand mark is a vine leaf with grape berries underneath, not a wine-glass icon from a stock pack. Category glyphs are bottle, barrel, glass, vine leaf — drawn rather than borrowed.
5. **Generosity of space.** Cabinet-magazine generosity in sidebar/section spacing. Cap reading widths at ~70ch. Whitespace is a design element, not a default.
6. **Bans (project-specific reinforcement of skill bans):** no border-left accent stripes, no gradient text, no glassmorphism, no black-and-gold luxury cliché, no cyan or purple anywhere, no "score" chrome (stars, badges, scales).

### Typography Commitments
- **Display**: *Caprasimo* — same as openculturemap, for visual continuity across the OHM family.
- **Body**: *Vollkorn* — generous, slightly-warm reading serif.
- **No monospace** anywhere in the UI. Numbers in the body face with `tabular-nums` where alignment matters.

### Color Commitments
- Use OKLCH everywhere.
- Light theme: warm paper background (~98.5% L, ~0.014 chroma toward 92° / parchment), ink with a red warm cast (~20% L, ~0.030 C, ~28° hue), claret primary accent (~42% L / 0.165 C / ~22° hue), gold secondary (~72% L / 0.140 C / ~88°), vine-green tertiary (~50% L / 0.110 C / ~145°), terracotta and plum reserved for the brand mark.
- Dark theme: deep cellar background (~16% L / 0.025 C / ~22°), aged-paper text, accents tuned upward in lightness and downward in chroma to read correctly on dark.
- Neutrals tinted toward claret, not pure gray.

### Cartography
- Light basemap: `tiles.openfreemap.org/styles/positron` — clean and quiet so the wine layer stays the hero.
- Dark basemap: CARTO dark-matter for warmth at low light.
- Cluster fill is the claret accent; cluster text is paper. Single-feature glyphs are tinted to their category color and rendered with a thick paper halo so they survive any basemap.
